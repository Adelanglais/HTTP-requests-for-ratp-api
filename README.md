# HTTP-requests-for-ratp-api

Code permettant d'obtenir des informations à partir d'une API.   
API utilisée :  ratp   
Format :  XML   

## Installation
```bash
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```
# Usage

```python
python3 ratp.py # returns next stop time of line 7
```
# Quit virtualenv 
```bash
deactivate
```
